# HATEDEMICS Models and Data

This repository contains resources developed within the HATEDEMICS project, including multilingual models and annotated datasets for hate speech and check-worthiness detection in English, Spanish, Polish, and Italian.

## Contents

- Annotated datasets for hate speech and check-worthiness detection
- Fine-tuned models for multilingual classification
- Scripts for preprocessing, training, and evaluation
- Documentation on labels, data format, and model usage

## Languages

The repository currently includes resources for:

- English
- Spanish
- Polish
- Italian

## Tasks

### Hate Speech Detection

Models and annotations for identifying hate speech in online content.

### Check-Worthiness Detection

Models and annotations for identifying claims or messages that may be relevant for fact-checking.
